4.7   Setting up inserters

  • Dashboards view

    Inserter devices let you supplement your printed output during the production run with inserts, such as pre-printed advertisements or return envelopes. You can set up inserter objects in the asset library to correspond to the contents of inserter devices.

    For example, suppose you print billing statements and they include both a return envelope and two pre-printed insert ads. You can set up an inserter object for each insert and inserter device configuration.

    A single inserter object can reference multiple bins on a single print device; or, if you prefer, you can also create separate inserter objects that all point to the same print device. After you have created an inserter object, you can attach a barcode to the inserter object to control when inserts are included.

    Creating an inserter

    1. In the Communication asset library, click the Delivery tab.

    2. Click Add and then click Inserter.

    3. In the Properties area, add a name and optionally a description.

      Names do not support the following special characters: & / : * ! ? > < | + # %

    4. Click Create.

    Defining inserter properties

    After you create the inserter object, you can use the Properties panel to view and modify the following properties.

    1. From the Number of bins list, select the number of inserter bins on your printer. You can define up to 32 inserter bins.

    2. Optional You can use a barcode to control bin selection so that only the intended inserts are included in a customer’s output. You can also use barcodes to include other information, such as using a variable that is set up to display the customer number, count the sheets in the document, or track other processing equipment information.

      To assign a barcode to an inserter:

      1. In the Barcodes area, click Add.

      2. In the Barcode dialog box, select a barcode. Only approved barcodes are available for selection.

      3. Click OK. The barcode is added to the inserter.

      4. To specify the location where you want to place the barcode in duplex output, select an option from the list adjacent to the barcode. Keep in mind that you can specify other barcode placement settings, such as the direction of the barcode, in the barcode properties. For more information about specific barcode properties, see Section 5.1 “Creating barcodes”.

    3. Inserters also include additional common settings. For information about these settings, see the following locations:

      • Domains – For information about defining the domains to which a resource is assigned, see Section 3.13 “Associating resources with domains”.

      • Workflow – For information about managing an approval workflow, see Section 3.6 “Managing approval workflows”.

      • References – For information about viewing the linked resources connected to a selected resource, see Section 3.3 “Viewing resources in asset library views”.
