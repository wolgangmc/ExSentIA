3.9.4   Replacing text in multiple content resources

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    If your deployment supports the Search text within content option, follow the instructions in this section to replace text in multiple content resources. You can replace text in the standard variation of a content resource or any of its alternate variations.

    Replacing begins with a list of content resources in the search results. Each resource contains matching text in one or more locations. The match locations can be in the standard variation or any alternate variations.

    In the results list, you select the content resources in which to replace matching text. Then, you can either replace the matches at all locations in the selected resources or open individual resources and replace matches at selected locations.

    After replacing, you can review the final replacement results and the replacement status for each affected content resource.

    Prerequisites for replacing text in multiple content resources

    You can replace text in a list of content resources, including text in alternate variations of those resources, if each resource meets these requirements:

    • The advanced search option Search text within content is available for all content types in list that you want to search. For the list of content types that support the Search text within content option, see Applying the Search text within content option.

    • The text to replace is within an editable content resource. For more information about content editability requirements, see Section 3.8.2 “Prerequisites for replacing text while editing”.

    Note

    You cannot replace text in the name or description properties for content resources. However, you can search for text in the name or description properties if the Search text within content check box is cleared or unavailable.

    Finding and replacing text in multiple content resources

    1. To find content resources and replace text, complete the following steps:

      1. From the home page, access the list of content resources in which you want to replace text. You can replace text only in content resources that meet all the prerequisites for replacing text.

      2. In the Search box, click the down arrow.

      3. At the top of the Advanced search options dialog box, click Replace.

      4. Enter the search text in the Search... box and the replacement text in the Replace... box.

      5. Optional Select the check boxes for any advanced search options that you want to apply to your search results.

      6. At the bottom of the dialog box, click Replace to display the list of search results in the Content resource type text replacement dialog box.

        Note

        When searching for text to replace, text in the name and description properties for content resources is ignored.

    2. To choose how to replace search text in the results, complete the following steps:

      1. In the Content resource type text replacement dialog box, select the check boxes for the content resources in which you want to replace the search text

      2. Perform one of the following replacement tasks:

        Task Action
        To replace the search text in all locations in all selected content resources. Click Replace All.
        To open each selected content resource in the search bar and specify the locations at which to replace the search text Click Replace.

    3. To replace search text at selected locations using the search bar, complete the following steps:

      1. In the search results, if you clicked Replace instead of Replace all, the first selected content resource opens in an editng view with search bar and the matching text locations highlighted. The editing view initially shows the first match location at the top of the canvas, even if the first match is in an alternate variation.

      2. Perform any of the following actions in the search bar:

        Task

        Action

        To replace the search text in all locations in the content resource

        Click Replace All.

        To replace the search text at the current location and then go to the next location

        Click Replace.

        To skip replacing at the current location and go to the next location

        Click the click the Next button .

        To go to the next or previous search location

        Click the Next button or Previous button .

        To view the list of replacement results

        Click the Replacement list button . Click Return to go back to the search bar.

        To save replacements and move to the next selected content resource

        Click the Save changes button .

        To discard all replacements and move to the next selected content resource

        Click the Discard changes button .

        To abandon the remaining text replacements in the current resource and all remaing resources

        Click Stop.

      3. In the search bar, after you click Replace at the last location, or after you click Replace all at any location, do one of the following in the dialog box that displays the message: How would you like to advance to the next text object to replace matches?

        Task

        Action

        To save all replacements before advancing to the next content resource

        Click Save.

        To save no replacements before advancing the next content resource

        Click Abandon.

        To close the message box and return to the last location

        Click Cancel.

      4. Optional In the search bar, click Stop. Then, do one of the following in the dialog box that displays the message: Are you sure you would like to abandon remaining text replacements for all matches?

        Task Action

        To abandon all remaining replacements in the current content resource and all remaining content resources

        Click Stop.

        To close the message box and return to the last location

        Click Cancel.

    Reviewing replacement results

    1. After you save your last text replacement, or if you click Stop in the search bar and choose to skip any remaining text replacements, the Content resource type text replacement dialog box opens so you can review the following statistics in the Final results list:

      Result

      Description

      Matches found

      The number of search text locations in the content resources that you selected in the search results.

      Replaced

      The number of locations at which search text was replaced in the content resources that you selected in the search results.

      Ignored

      The number of locations at which search text was not replaced in the content resources that you selected in the search results.

      Could not change

      The number of content resources that you selected in the search results that would not allow text to be replaced. This number includes content resources that are selected in the search results, but are not in the draft state or are locked by another user.

    2. Click Close.

    3. In the Replacements found list, review the icons in the Replaced column for each selected content resource in the results. The icons indicate the replacement status for each content resource as follows:

      Icon

      Replacement status

      None: This status is due to one of the following conditions:

      • The resource was opened, but no replacements were made.

      • Text in the resource could not be changed because the resource did not meet the prerequisites for replacing text.

      Partial: The search text was replaced in at least one location, but not in all locations.

      All: The search text was replaced in all locations.

      Blank (no icon)

      Unavailable: The resource appears in the search results (contains matching text), but it was not selected for the replacement operation.
