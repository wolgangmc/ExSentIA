4.6.4   Using system variables

  • Monitor view

    System variables let you use system information, such as the information stored in the design environment, to complete tasks such as monitoring and controlling an engine run, formatting designs, or organizing a customer mailing.

    While some DAS system variable properties are predefined, you can edit the Design sample and Output format fields in the Properties panel. You cannot delete a system variable.
