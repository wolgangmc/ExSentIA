2.3.3.4   Configuring settings for a workflow type in a domain

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    1. In the OpenText Communications web client, access the Tenant management view.

    2. In Tenant management view, click the Domains tab, and then click a domain name in the tab.

    3. In the Properties panel for the domain, click the Workflow tab.

      If the Workflow tab is hidden, select the Use default workflow check box to show the Workflow tab.

    4. In the Workflow tab, choose the workflow type for the domain in the Workflow type list.

    5. Use the following table to configure the settings for your selected workflow type:

      Workflow type

      Settings

      Standard

      In the Workflow tab, select or clear the Require separate approval check box. If you select this check box, approvers cannot approve the drafts that they submit.

      For more information about the standard workflow, see Section 3.6 “Managing approval workflows”.

      Simple

      None.

      For more information about the simple workflow, see Section 2.3.3.5 “Using the simple workflow in a domain”.

      Advanced

      Configure approval role sets, resource types, approver groups, and other advanced workflow settings.

      For more information about configuring advanced workflow settings, see Section 2.3.3.6 “Configuring advanced workflow settings for a domain”.

    6. Click Save.
