    The Orchestration asset library view lets you manage and configure the orchestration resources that are used to generate template previews for Content Author and communication previews for Designer and Content Author. For information about creating and managing orchestration resources to run engine orchestration jobs and produce customer output, see Running engine orchestration.

    Note

    If you are using the Cloud Platform, Orchestration asset library view is not available if the OpenText Communications subscription is set to Base.

    The following table describes the different types of template and communication previews and lists the web client views that are used to access each preview type.

    Types of previews

    Preview type Description

    Template preview

    Lets you preview the frames in each template that are available for use in Content Author.

    You can preview templates in the following views:

    • Communication launcher view

    • Orchestration asset library view

    Note

    You can preview a template only if the following criteria are met:

    • The template contains Content Author frames.

    • The template includes more than a single document.

    • The template was not packaged in the Design Manager application with the option to split communications in Content Author.

    For more information about template previews, see Previewing Content Author templates.

    Communication preview

    Lets you preview the current communication, including all of the content that will appear in customer output.

    You can preview communications in the following views:

    • Communication view (Content Author only)

    • Simulation view

    Communication previews that are generated in Simulation view are also called simulations.

    For information about previewing communications and generating simulations in Content Author, see OpenText Communications - Content Author User Help (CCCA250100-H-UGD)OpenText Communications - Content Author User Help (CCCA250100-H-UGD).

    For information about generating simulations in Designer, see OpenText Communications - Designer User Help (CCCD250100-H-UGD)OpenText Communications - Designer User Help (CCCD250100-H-UGD).
