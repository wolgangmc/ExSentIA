8.3.2.4.2   Previewing content

  • Dashboards view

    When you select a text fragment, image fragment, variable, or rule in the Fragments panel, a preview of the content is highlighted in the Preview panel, and the information about the content opens in the Properties panel. If you select multiple fragments, the preview highlights all the selected content.

    StreamServe considerations:

    • If more than one StreamServe communication is included in a migration set, you can reduce the list of fragments by selecting a specific communication in the Select a communication list in the Fragments panel.

    • If a StreamServe template-based migration set does not include any communications or themes, the Preview panel displays a preview of the template.

    You can scroll up and down in the Preview panel to see the pages within a PDF, DOCX, or template. If there are multiple occurrences of a fragment, you can use the arrow buttons on either side of the preview to navigate between occurrences.
