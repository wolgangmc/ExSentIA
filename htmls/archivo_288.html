10.4   Resource workflow widget

    The Resource workflow widget displays a list of selected DAS resources and their respective workflow states.

    To add a resource workflow widget:

    1. Click the Dashboards button.

    2. Click Add widget.

    3. Enter a name and, optionally, a description for the widget.

      Names do not support the following special characters: & / : * ! ? > < | + # %

    4. In the Widget type list, select Resource workflow.

    5. In the Resource library list, select which groups of resources you want to see based on their DAS library.

    6. In the Workflow list, select the resource workflow states you want to include.

    7. In the Resource type list, select the resource objects that you want to include. This list changes based on the Resource library options you selected.

    8. If the tenant includes category tags, you can filter the list of resources by selecting one or more tags in the Category list.

    9. Click Create.

      After the widget is added, the dashboard changes to edit mode.

    10. Click the Dashboards button.

    11. Click Save.

    After the widget is populated with resources, you can use it to move selected resources through the approval workflow states.

    Tip

    • To open a resource, double-click the entry in the widget.

    • If a resource has child resources, you can view them by clicking the Expand button at the end of the row. Selecting the parent resource automatically selects all child resources.

    • If a resource includes a workflow comment, a message icon appears next to the workflow symbol; for example: . Hovering over the icon displays the comment.
