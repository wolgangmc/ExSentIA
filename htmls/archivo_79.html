2.4.3   Setting the default font and list formats for Designer

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    For communications based on a communication set, you can set the default font to be used when creating designs in Designer, as well as the default font for footnotes. You can also set the default bullet and numbering formats used for up to four levels of indentation. These defaults will apply to all supported designs created in the tenant.

    Only approved fonts that have been added in the Design asset library view are available to be set as default fonts. If also you want the default font to be available in Content Author templates, you must enable that option in the font properties.

    If a design has already been created using other fonts, the default font will only be applied to content created after the default is set. Likewise, if you change the default font, the new default applies only to text entered after the font change.

    Similarly, bulleted or numbered lists will only use the default format for lists created after the default is set.

    Note

    If default bullets are not defined, when a user creates a bulleted list in Designer they will be prompted to contact their administrator to set defaults. However, they will be able to define and use temporary bullet characters.

    To set the default font and list formats:

    1. In the OpenText Communications web client, sign into the following URL as a tenant administrator user:

      https://<host>:<optionalPort>/design?tenant=<tenantName>

    2. To open Tenant management view, click the Tenant management button .

    3. In the Configurations panel, click Designer.

    4. In the Properties panel, select the Text and Fonts tab.

    5. Under Default font, click Set default font and select the font that you want to use.

      You can also change the font size, and, if available, apply bold or italic styling.

    6. Under Bullets, you can define the bullet symbol and color for each bullet level.

      To select the bullet symbol:

      1. Click the box next to the level that you want to change to open the Bullet symbol selection dialog box.

      2. If you want to change the font used for the bullet, select an approved font in the Typeface list.

      3. Select the symbol to use as the bullet.

      4. Click Set.

      To change the bullet color:

      1. Click the color block next to the bullet box.

      2. Use the color selection dialog box to select the bullet color.

      3. Click Apply.

    7. Under Numbering, you can select the numbering style and color for each number level.

      To change the type of numbering that is used for a particular level, click the arrow next to the numbering example and select the style that you want to use.

      To change the color that is used for the number:

      1. Click the color block next to the number box.

      2. Use the color selection dialog box to select the number color.

      3. Click Apply.

    8. Under Default footnote font, click Set default font and select the font that you want to use inside footnote frames.

      You can also change the font size, and, if available, apply bold or italic styling.

    9. Click Save.
