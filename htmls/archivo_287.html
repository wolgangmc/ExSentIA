10.3   Job list widget

    The Job list widget can display a list of jobs or flow instances along with their ID, status, and status message, if applicable. The data can be filtered by flow model, input channel, status, or timeframe.

    To add a job list widget:

    1. Click the Dashboards button.

    2. Click Add widget.

    3. Enter a name and, optionally, a description for the widget.

      Names do not support the following special characters: & / : * ! ? > < | + # %

    4. In the Widget type list, select Job list.

    5. In the Widget style list, select an initial size for the widget based on the number of tiles it covers. Options include Large (2x2) and Tall (1x4).

      After you create the widget, you can customize its size by dragging the edges of the widget. Changing the size of a widget automatically changes its style, as well. For example, changing a 2x2 widget to 1x1 alters the style from Large to Small. If you want to resize the widget but do not want the style to change, then select the Lock widget style check box.

    6. In the Filter by list, select whether to show flow instances or jobs.

      • If you select Flow instances, use the Flow model list to specify which jobs are visible, based on their associated flow models.

      • If you select Jobs, use the Input channels list to specify which flows are visible, based on their associated input channel.

    7. In the Status list, select whether to include or exclude one or more of the following job statuses: Canceled, Complete, Deleted, Error, Holding, Started, or Waiting.

    8. In the Timeframe list, select one of the following options: Date-time range, Recent, or Today.

      • If you select Date-time range, use the From and To boxes to enter a range of dates. You can also click the calendar button to select a date instead of manually entering it.

      • If you select Recent, designate the number of minutes, hours, or days.

    9. Optional If you want the widget data to automatically update, select the Auto refresh check box and then specify the time interval in the Auto refresh interval list.

    10. To specify how the widget should display the results, use the Sort by and Sort direction lists.

    11. Click Create.

      After the widget is added, the dashboard changes to edit mode.

    12. Click the Dashboards button.

    13. Click Save.

    Tip

    To open a job or flow instance entry in Monitor view, double-click the entry in the widget.
