2.3.2.8   Deleting a domain

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    To delete a domain, select the domain in the Domains panel in Tenant management view, and then in the Properties panel, click Delete.

    Note

    To successfully delete a domain, make sure that the domain does not contain any resources and no roles are currently assigned to that domain.
