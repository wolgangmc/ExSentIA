3.8.1   Searching for text while editing

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    1. In the editing view, do either of the following:

      • On the toolbar, click the Search button . Then, in the Search for box on the search bar, enter the text that you want to find.

      • In a content canvas, select the text that you want to find. Then, on the toolbar, click the Search button .

      All search results in the content canvas, or canvases, are underlined in red.

      For advanced search options, see step 2.

      Tip

      Before searching in Section view, see Section 3.8.5 “Guidelines for searching and replacing text in Section view”.

    2. Optional To perform a case sensitive or whole-word search:

      1. In the search bar, in the Search for box, click the Search options arrow.

      2. Select one or both of the following advanced search options:

        Advanced search option Description
        Case sensitive search Limits search results to text that matches the case of letters in the search text.
        Whole word search Limits search results to text that matches whole words in the search text. For example, a whole-word search for the word license would find license, but not licenses, licensed, or licensee.

      3. Click Apply

    3. Optional In the search bar, click the Next button or Previous button to highlight and select the next or previous search result. The numbers beside the Search for box tell you which search result is selected and the total number of results, for example . If you select the last search result and click the Next button , the selection highlight returns to the first result.

    4. Optional Enter new search text in the Search for box to start another search.

    5. Click Stop to close the search bar.
