2.5.3   Enabling OpenText Core Journey integration

    OpenText Core Journey allows organizations to map customer journeys and track customer activity for every event, across all customer journey touchpoints. The solution helps identify which events are working and which need improvement. To let users view event data for OpenText Communications communications, you must integrate your OpenText Communications environment with OpenText Core Journey. Event data, which is sent to OpenText Core Journey as a JSON file, can include information such as the number and types of documents that are generated by OpenText Communications and user-interaction information for output delivery channels.

    To enable OpenText Communications to send data to OpenText Core Journey:

    1. Use a Helm command to enable the OpenText Core Journey integration service within the OpenText Communications deployment. For information about deploying OpenText Communications, see OpenText Communications - Cloud Deployment Guide (CCMCE250100-IGD)OpenText Communications - Cloud Deployment Guide (CCMCE250100-IGD).

    2. In Tenant management view in the OpenText Communications web client, configure OpenText Core Journey integration for each domain from which you want to send data. For more information, see Configuring OpenText Core Journey integration.

    3. In Orchestrator, enable document tracking for communications and output channels. For more information about enabling tracking, see OpenText Communications - Orchestrator User Help (CCMCEORC-H-UGD)OpenText Communications - Orchestrator User Help (CCMCEORC-H-UGD).

    By default, the event cron job for the Journey integration in your deployment runs every 10 minutes to send event data from OpenText Communications to OpenText Core Journey. You can change this frequency by using a Helm command to set the ei.schedule.event parameter. For more information, see OpenText Communications - Cloud Deployment Guide (CCMCE250100-IGD)OpenText Communications - Cloud Deployment Guide (CCMCE250100-IGD).

    To visualize the data points that are sent from OpenText Communications, you can create customer journeys in OpenText Core Journey. For more information about creating customer journeys, see OpenText Core Journey User Guide (CDPEXPCDI-UGD).

    Note

    For information about OpenText product compatibility, see the OpenText compatibly information provided in the release notes for the product. Release notes are available on My Support.
