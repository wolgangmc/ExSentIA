3.8   Searching and replacing text while editing

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    In an editing view, such as Design view or Section view, you can search and replace text while editing a content resource. Search results can be found the main text of the resource and in any objects that it contains.

    Searches can include text in editable and non-editable content objects, but you can replace search results only in editable objects. For more information about replacing text, see Section 3.8.2 “Prerequisites for replacing text while editing”.

    Whether you are editing a single piece of content in a single canvas or multiple pieces of content across multiple canvases, you can replace search results selectively or all at once. If you replace results selectively, then you can undo replacements until you save the current canvas. However, if you replace all results at once across multiple canvases, then the replacements are automatically saved in each canvas and the replacements cannot be undone.

    Note

    To search and replace text across multiple, text-based content resources when you are not editing, see Section 3.9 “Searching and replacing text in multiple content resources”.
