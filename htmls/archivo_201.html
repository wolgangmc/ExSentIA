5.12   Adding PDFs

  • Communication asset library view

  • Orchestration asset library view

  • Dashboards view

    You use the Common panel in the Design asset library view to add PDF resources to the asset library. PDFs that are added to DAS can be used in Designer to import text and graphics into a page design.

    To add a PDF:

    1. In the Common panel, click Add, then click PDF in the selection list.

    2. In the file browser, select the PDF that you want to add, then click Open.

    3. Optional Select the newly added PDF in the Common panel and on the General tab in the Properties panel, enter a description to help distinguish it from other files and assign categories, if available.

    When you add a PDF to the library, it is in the draft state. To use a PDF with a page design, the PDF resource must be approved. Unapproved PDF resources are not visible in the PDF browser in Designer.
