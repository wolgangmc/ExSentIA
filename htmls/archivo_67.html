2.3.3.2   Showing or hiding workflow settings in domains tenant-wide

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    1. In the OpenText Communications web client, access the Tenant management view.

    2. In Tenant management view, click Configurations > Content Author.

    3. On the Content Author properties panel, in the General tab, do one of the following:

      • Select the Use default workflow check box to show the Workflow tab on the Properties panel for each domain, tenant-wide. The Workflow tab in each domain contains its workflow settings.

      • Clear the Use default workflow check box to hide the Workflow tab tenant-wide.

      Note

      Selecting or clearing the Use default workflow check box also enables or disables the default workflow in Content Author tenant-wide. However, the default workflow remains enabled tenant-wide in Designer and in the asset library views after you clear the Use default workflow check box.
