3.9.2   Specifying advanced search options

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    If you want to limit or refine the search results when you search or replace text in content resources, specify advanced search options.

    To specify advanced search options:

    1. In the Search box, click the down arrow.

    2. In the Advanced search options dialog box, specify search text and, if necessary, replacement text. Then, select the check boxes for one or more of the following options:

      Option

      Effect

      Case sensitive search

      Limits results to content resources that contain text that matches the case of letters in the search text.

      Show unavailable content

      Extends results to include content resources that you cannot access. For example, the results can include older versions of content resources and resources that are reserved by a different user.

      Search text within content

      Limits results to content resources that contain editable text that matches the search text. For example, if the search text is a word or phrase, the results can include messages or clauses that contain the word or phrase.

      For more information, see Applying the Search text within content option.

      Note

      Clicking Replace automatically selects the Search text within content check box. The Replace button appears only if the Search text within content option is available. For more information, see Section 3.9.4 “Replacing text in multiple content resources”.

      Workflow state

      Limits results to content that matches the selected workflow state or states: Draft, Submitted for approval, Approved, or Rejected.

      Quick tags

      Limits results to content that has the selected category tags or metadata (class) tags applied.

      My objects

      Limits results to content that you created, reserved, or last updated.

      Include text within fragments

      Limits results to fragments that contain text that matches the search term.

      Note

      This option is only available in the Rationalization user interface and is enabled by default.

      Note

      The available advanced search options and the ability to replace text in multiple content resources depend on where you access the Search box, and whether the Search text within content option is enabled. If you do not see the Search text within content check box, then ask your administrator about adding this option.

    Applying the Search text within content option

    If the Advanced search options dialog box includes the Search text within content check box, you can select this option to search and optionally replace the editable text within the following types of content resources:

    • Text message

    • Graphic message

    • Clause (CA or CD)

    • SMS message (CA or CD)

    • Page

    • Email

    • Web

    • Component

    • Design layout

    If you search in the Design asset library view, then the Search text within content option is available only if the list of content resources in the view contains only the types above.

    Selecting or clearing the Search text within content check box affects searches as follows:

    Status Action

    Selected

    • Searches include the editable text within text-based content resources and their variations.

    • Searches exclude text in resource names and descriptions.

    • Search results can be replaced.

    Cleared or unavailable

    • Searches include the text in resource names and descriptions.

    • Searches exclude editable text within text-based content resources and their variations.

    • Search results cannot be replaced.

    Note

    If you do not see the Search text within content check box, then ask your administrator about adding this option.
