4.2.3.4   Changing an existing variable mapping

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    If you have previously mapped a variable to a data area, but would like to remove that mapping and map that variable to a different data area, simply drag and drop the variable chip to the other data area. Any data area properties that are associated with that variable will revert to their default settings after the variable has been moved.
