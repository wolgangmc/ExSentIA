3.9.1   Specifying the text to find or replace

    To search or replace text across multiple resources, start by specifying the search text. Then, if needed, specify the replacement text. The search text or replacement text can be a word, a phrase, or a sequence of characters.

    To specify the search text or replacement text, in the Search box, do the following:

    Task

    Action

    To specify search text

    Enter the search text. Or, click the down arrow to open the Advanced search options dialog box, and then enter the search text in the Search... box.

    To specify replacement text

    Click the down arrow to open the Advanced search options dialog box, and then enter the replacement text in the Replace... box.

    Search text and replacement text can contain letters, numerals, spaces, and any of the following characters:

    ~ ` @ $ ^ ( ) _ - = ; " ' , .

    Search text and replacement text cannot contain any of these characters:

    ! # % & * + | \ / <> : ?

    Tip

    If you enter invalid search text or replacement text, a red outline appears around the text and you cannot search or replace.

    You can put quotation marks around the search text to change the type of text matching that occurs in content resources. The following table shows the results if you enclose the search text in quotation marks, and the results if you do not:

    Search text in quotation marks Results
    Yes The search returns content resources that contain one or more instances of the whole search text. The words in each instance are in the same order as the words in the search text.
    No The search returns content resources that contain one or more instances of every word in the search text, in any order.

    Note

    In search text, a word is any string of one or more consecutive, valid characters, including dictionary words. Spaces separate words in the search text.

    If you are replacing text, then the search text in the Search... box and the replacement text in the Replace... box are both enclosed in quotation marks that cannot be removed. As a result, a Replace operation finds and replaces only text that matches the whole search text.
