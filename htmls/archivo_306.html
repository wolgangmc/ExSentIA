  • Dashboards view

    Dashboards view provides the ability to create multiple domain-specific dashboards that contain widgets. The widgets are used to monitor various aspects of the domain, such as orchestration job results and transaction usage. Dashboards can be shared with other roles and groups that have permission to access this resource.

    The following widgets are available in Dashboards view:

    • Job status—Displays a graph of the jobs run through orchestration.

    • License—Provides information and status about the license for the tenant.

    • Job list—Provides information about the status of selected flow models.

    • Resource workflow—Lists specified resource types and their workflow states. Also provides the ability to move resources through the approval workflow states.

    • Output review—Lists all the flow steps which are on hold and provides a preview of the output.

    To create a new dashboard:

    1. Click the Dashboards button .

    2. In the Dashboards panel, click the Dashboards button .

    3. Click Create.

    4. Enter a name for the dashboard.

    5. Click Create.

    6. Click the Dashboards button again.

    7. Click one of the following:

      • Add widget—Create a widget.

        Refer to the individual widget descriptions for information about their options.

      • Cancel—Exits edit mode.

    Once a dashboard has been created and saved, the following options become available when you click the Dashboards button:

    • Add widget—Create a widget.

    • Create—Create a new dashboard. Optionally, you can create a new dashboard by cloning an existing one.

    • Edit—Changes the dashboard into edit mode, allowing you to move, resize, rename, delete, and update filters for existing widgets, as well as add widgets.

    • Refresh—Refresh the active dashboard. Note that widgets do not auto-refresh.

    • Delete—Delete the active dashboard.

    • Manage—Rename, delete, or rearrange the order of existing dashboards. Note that the dashboard order is a local browser setting; using a different browser or clearing the browser cache resets the order to arrange by name.

    Tip

    After a widget has been created and saved, you can temporarily update the filter settings for a widget by clicking the Apply filter button .
