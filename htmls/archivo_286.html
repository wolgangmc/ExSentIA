10.2   License widget

    The License widget displays the number and percentage of transactions and messages used in the license for the current tenant, as well as the named user seat usages.

    Keep the following things in mind when using a license widget:

    • To view license information, the OTDS group which is associated with the OpenText Communications role needs to be a member of one of the OTDS administrative groups. For more information about OTDS group assignments, see OpenText Directory Services Web Client Help (OTDS-H-AWC).

      Note

      For information about OpenText product compatibility, see the OpenText compatibly information provided in the release notes for the product. Release notes are available on My Support.

    • Because OpenText Communications deployments that use a non-production product key do not count transactions, this widget is most relevant for production domains.

    • Because widgets do not auto-refresh, there may be discrepancies between the OpenText Communications dashboard and the OTDS transaction usage reporting. For OpenText Compliance purposes, refer to usage information for the OTDS tenant managing the license for the relevant OpenText Communications tenant.

    To add a license widget:

    1. Click the Dashboards button.

    2. Click Add widget.

    3. Enter a name and, optionally, a description for the widget.

      Names do not support the following special characters: & / : * ! ? > < | + # %

    4. In the Widget type list, select License.

      Note

      In the Widget style list, Medium (1x2) is the only option. After you create the widget, you can customize its size by dragging the edges of the widget.

    5. Click Create.

      After the widget is added, the dashboard changes to edit mode.

    6. Click the Dashboards button.

    7. Click Save.
