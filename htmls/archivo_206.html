  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    Although the following objects are typically created and defined from Designer view for Content Author, they are also stored in the web client on the Author tab of the Design asset library. From the web client, you can review the settings for these objects and define how these objects behave when they interact with customer data and output.

    • Clause (CA)

    • Communication (CA)

    • Graphic message

    • Messaging (CA)

    • Section (CA)

    • Text message

    To avoid unexpected output changes, the settings on the General tab are typically view-only. However, you can modify the following settings as needed, depending on the object you are viewing:

    • Domains – For information about defining the domains to which a resource is assigned, see Section 3.13 “Associating resources with domains”.

    • Workflow – For information about managing an approval workflow, see Section 3.6 “Managing approval workflows”.

    • References – For information about viewing the linked resources connected to a selected resource, see Section 3.3 “Viewing resources in asset library views”.
