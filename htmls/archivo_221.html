8.2.1   Creating Content Author communication containers and documents

    In addition to Content Author communications, you can create the following types of Content Author resources in DAS:

    • Content Author communication containers

    • Content Author documents

    Content Author communication containers include one or more Content Author documents. You create communication containers and their documents from container templates that have one or more document templates.

    A container requires at least one document. When you create a new container, you select a container template and select one of its document templates to create the first document. You can add more documents to the container later if the container has unused document templates.

    Each document template can create one document in each container that it is associated with. After you create a document in a container, the document’s template is no longer available in that container.

    For more information about creating containers and adding documents to containers, see section 1.1.2.1 "Creating a communication container with documents" in OpenText Communications - Content Author User Help (CCCA250100-H-UGD)section 1.1.2.1 "Creating a communication container with documents" in OpenText Communications - Content Author User Help (CCCA250100-H-UGD).

    To list containers and documents in your deployment

    1. In Communication launcher view, click the Communications panel.

    2. In the Communication name column, look for names with a container icon .

    3. Click a container name in the list to show or hide a tree view of documents in the container.

    4. Optional Do the following to see a list of container assets and document assets:

      1. In Design asset library view, click Author > Library.

      2. In the Filter assets list, select Communication (CA).

      3. In the Type column, look for Container and Document assets.

    To list container templates and document templates in your deployment:

    1. In Communication launcher view, click the Communications panel.

    2. Click New.

    3. In the Template name column, look for names with a container template icon .

      The Template name column includes only approved templates.

    4. To see the list of document templates that are available in a container, create a new container or create a new document in an existing container.

      For more information about creating containers and adding documents to containers, see Section 8.2.4 “Starting a Content Author communication container”.
