3.9   Searching and replacing text in multiple content resources

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    Several places in your deployment display lists of content resources, along with a Search box. You can use the Search box to search, filter, and in some cases replace text in the listed resources.

    By default, searches include only the text in content resource names and descriptions. However, if the Search text within content option is enabled, you can instead search and replace text within text-based resources such as messages or clauses.

    Note

    You can also search and replace text while editing a text-based resource. For more information, see Section 3.8 “Searching and replacing text while editing”.
