8.3.2.2.6   Exporting and importing migration sets

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    Migration sets can be exported individually using the same method as other DAS resources.

    When you export any of the following resources that have an associated migration set, a separate export package ZIP file is created for each associated migration set:

    • Data source

    • Communication set

    • Flow context

    • Designer communication

    • Content Author communication

    To import a migration set package ZIP file, you must use the Rationalization import format type.
