3.8.5   Guidelines for searching and replacing text in Section view

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    When you search or replace text in Section view, keep the following guidelines in mind:

    • You can replace text in any editable clauses in a section, even if the section is not editable. For more information, see Section 3.8.2 “Prerequisites for replacing text while editing”.

    • You can search and replace text only in the selected section. Other sections, such as nested, peer, or parent sections, are not included. To search and replace text in another section, open the other section in a separate Section view.
