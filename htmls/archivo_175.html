4.10   Setting up paper bins

    For print-based content, you can store and use multiple paper bin configurations. Each paper bin must be set up as a separate object in the asset library.

    Creating a paper bin

    1. In the Communication asset library, click the Delivery tab.

    2. Click Add and then click Paper bin.

    3. In the Properties area, add a name and optionally a description.

      Names do not support the following special characters: & / : * ! ? > < | + # %

    4. Click Create.

    Defining paper bin properties

    After you create the paper bin object, you can use the Properties panel to view and modify the following properties.

    1. From the Number of paper types box, select the number unique paper types to include. You can select up to eight different paper types using one paper bin object.

    2. From the paper type selection boxes, select the paper type you want to use for each bin.

    3. Paper bins also include additional common settings. For information about these settings see the following locations:

      • Domains – For information about defining the domains to which a resource is assigned, see Section 3.13 “Associating resources with domains”.

      • Workflow – For information about managing an approval workflow, see Section 3.6 “Managing approval workflows”.

      • References – For information about viewing the linked resources connected to a selected resource, see Section 3.3 “Viewing resources in asset library views”.

    Adding paper bins to an output

    1. In the Communication asset library, click the Delivery tab and select an output that supports paper bins to open it in the Properties area.

    2. Click the Paper bins tab.

    3. Select the Use control bin selection check box.

    4. Click to add a paper bin.

    5. Select one or more paper bins to apply to your output.

      • This option is available only if an approved paper bin object is available in the asset library.

      • Only paper bins that are compatible with your output appear in the list.

    6. Click OK.

    7. Click Save.
