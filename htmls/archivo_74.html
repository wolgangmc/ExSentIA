2.4.1.1   Creating configurations for Designer

    You can use the Configurations panel in Tenant management view to make the following configurations for Designer:

    On the General tab:

    Effectivity

    You can select the Use effectivity check box if you want to let designers apply effective dates to documents, designs, and components in Designer. You can also set effective and expiration dates on documents, designs, and components in Design asset library view; however, designers will be able to change that setting later on in Designer.

    Data Platform

    You can select the Data Platform check box to give designers access to the Experience Customer Data Platform (CDP) tools. To enable this functionality in Designer, you must specify a URL to connect to OpenText Customer Data. You can use the Run validation button located below the Data Platform URL box to check that the OpenText Customer Data URL is properly connecting to the service.

    Core Signature Services

    You can select the Enable Core Signature designs check box to allow designers to include signature objects in designs.

    Allowed URLs for previews

    In the Allowed script and HTTP URLs for preview box, enter the URLs for the sources of HTML code that contains external requests, such as scripts or calls to API endpoints. OpenText Communications is permitted to access the URLs that you add to this box when you preview a communication in Communication view or Simulation view. Separate URLs with a comma or semicolon.

    On the Text and Fonts tab:

    For communications based on a communication set, you can specify the following settings:

    • The default font for creating designs in Designer

    • The default bullet and numbering formats for up to four levels of indentation

    • In the Resource naming convention area, designate a character prefix to use when naming images and fonts in AFP output. You can use either one or two characters.

      The resource naming convention for image and font names in AFP output. This is a one- or two-character prefix appended to the names.

    • The default font for text within footnote frames

    These defaults will apply to all supported designs created in the tenant.

    On the Dictionaries tab:

    You can use the settings on the Dictionaries tab to manage installed spelling dictionaries for Designer.
