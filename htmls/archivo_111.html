3.7   Searching and replacing text

    You can search and replace text while editing or while managing lists of text-based content resources, such as messages or clauses. You can also apply filters to your search if you want to find text only in resources with specific properties.

    Differences: Searching while editing and searching across multiple resources

    The differences between searching and replacing text in an editing view and searching and replacing text starting from a resource list view are summarized as follows:

    • To start a search:

      • In an editing view: Click the Search button in the editing view toolbar and enter the search text.

      • In a resource list view: Click in the Search box for the list and enter the search text.

    • To select search filters:

      • In an editing view: In the search bar, click the arrow in the Search for... box to open the Advance search options list.

      • In a resource list view: Click the arrow in the Search box to open the Advance search options dialog box.

      In both cases, you select the filters to apply before you search or replace. However, you have several more filters to choose from when searching across multiple resources. For more information about selecting filters when you search across multiple content resources, see Section 3.9.2 “Specifying advanced search options”.

    • To review search results:

      • In an editing view: In the search bar, click Search. All results a highlighted in the editing canvas, or in multiple canvases if you are searching across variations, versions, or clauses in a section.

      • In a resource list view: Click the Search button. The list updates to show only resources that contain the search text, and that match the advanced search filters.

    • To replace search text:

      • In an editing view: In the search bar, enter the search text and replacement text, and then click Replace or Replace all.

      • From a resource list view: In the Search box, open the Advanced search options dialog box, enter the search text and replacement text, and then click Replace. Next, perform the following steps:

        1. In the results dialog box, select the resources in which to replace text, and then click Replace or Replace all.

        2. In each successive editing view for a selected resource, review the highlighted search results and replace matches.

        3. Save or abandon replacements to move to next selected resource.

        4. Review the replacement statistics for the selected resources.

    For more information about searching and replacing text in an editing view, see Section 3.8 “Searching and replacing text while editing”.

    For more information about searching and replacing text starting from a resource list view, see Section 3.9 “Searching and replacing text in multiple content resources”.
