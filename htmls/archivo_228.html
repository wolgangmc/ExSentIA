  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    With Content Author, in addition to communications, you can access the following types of content from the Communication launcher view.

    • Messages—Including text and graphic messages

    • Clauses

    • Sections

    • SMS—SMS messaging content

    Note

    • If you use Core Communications, then the SMS tab might not be visible in your configuration. For information about which functions are available as part of your subscription, speak to your administrator.

    • Users that have only Approver permission initially see only content in the Submitted for approval workflow state in Communication launcher view. If those users want to see content in other workflow states, they must apply an advanced search filter that has the other states selected. For more information, see Specifying advanced search options.
