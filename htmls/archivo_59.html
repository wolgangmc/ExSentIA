2.3.2.3   Modifying role assignments for an assigned OTDS group

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    1. In the Domains panel in Tenant management view, select the domain for which you want to define user permissions.

    2. Go to the Group assignments tab in the Properties panel.

    3. In the Currently assigned groups area, click the Role list for the OTDS group for which you want to modify role assignments.

    4. In the Role list, select or clear role assignment check boxes as needed.
