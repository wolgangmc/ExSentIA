3.1   Supported DAS resource types

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    The following table describes the resource types that are supported in DAS and lists the asset library view that you use to manage them.

    Table 3.1: Supported resource types

    Resource type Description Asset library view

    Alert model

    A resource that lets you create an alert flow based on an orchestration job state change. Alert flow models contain nodes that receive the alert from an alert node, then they process the rest of the flow like a regular flow model.

    Alert models are configured in Orchestrator.

    Flow panel in Orchestration asset library view

    Barcode

    A barcode object that can be used in Designer communications.

    Common panel in Design asset library view

    Clause

    A design object that is used in Content Author communications.

    Author panel in Design asset library view

    Color

    A design object that defines an individual custom color.

    Common panel in Design asset library view

    Color family

    A design object that is a color palette containing the colors that designers can use while creating content. A color family can contain any number of color objects.

    Common panel in Design asset library view

    Communication (CA)

    A Content Author communication, which is based on an application template that can be created from Design Manager applications or from Designer communications.

    For information about other types of Content Author communication resources, see Section 8.2.1 “Creating Content Author communication containers and documents”:

    Author panel in Design asset library view

    Communication (CD)

    A Designer communication, which can contain documents and designs such as pages and emails.

    Designer panel in Design asset library view

    Communication set

    A collection of other communication assets such as data sources, variables, and output queues.

    Communication sets can be used to start new Designer communications with DAS-only resources instead of using a template based on a Design Manager application.

    Communications sets panel in Communication asset library view

    Component

    A design object that can be reused in Designer designs.

    Designer panel in Design asset library view

    Data source

    A mapping of variables to a sample data file.

    When producing customer output, a data source is used to identify data areas in production data files.

    Data panel in Communication asset library view

    Design layer

    A set of static components that can be used to assist with design creation in Designer designs.

    Common panel in Design asset library view

    Design layout

    A Designer design template which can be used with page designs.

    Designer panel in Design asset library view

    Design pack

    Contains Design and Production resources that can be used in Content Author and Designer. Design packs are created in Design Manager and uploaded to the asset library along with the associated package file.

    Design packs panel in Orchestration asset library view

    Document

    A container object for designs in a Designer communication.

    Note

    This Document resource in Designer is different than a document in a communication container in Content Author.

    Designer panel in Design asset library view

    DOCX

    A DOCX file that can be imported into DAS so that its contents can be used in Designer communications.

    Common panel in Design asset library view

    Email

    A type of design layout that is used in Designer documents to create designs for HTML Email output.

    Designer panel in Design asset library view

    Event Events are snippets of code in a communication that execute at specific points during an engine run. Events that are saved as library events in Designer appear as DAS resources.

    Data panel in Communication asset library view

    Flow context

    Contains input and output channels and communication settings that are used to set up flow models.

    Flow contexts are configured in Orchestrator.

    Flow panel in Orchestration asset library view

    Flow model

    Defines the flow for engine orchestration jobs. Flow models contain nodes that receive data from an input channel, process the input data, and deliver the processed data through appropriate output channels.

    Flow models are configured in Orchestrator.

    Flow panel in Orchestration asset library view

    Flow script

    A resource that lets you create reusable flow scripts in order to implement orchestration logic.

    		Flow panel in Orchestration asset library view

    Font

    A font object that can be uploaded to DAS and used in Content Author or Designer communications. For communications that are based on Design Manager applications, DAS fonts supplement the fonts that are included in the design pack.

    Common panel in Design asset library view

    Graphic message

    A design object that is used in Content Author communications.

    Author panel in Design asset library view

    HTML

    HTML code that can be applied to web designs in Designer.

    Common panel in Design asset library view

    Image

    Image that can be uploaded to DAS and used in Content Author or Designer.

    Common panel in Design asset library view

    Inserter

    A resource that defines the contents of inserter devices, which are used to supplement printed output.

    Delivery panel in Communication asset library view

    Language

    A resource object that defines a language and its settings.

    Region panel in Communication asset library view

    Locale

    A resource object that defines the settings for a geographic location.

    Region panel in Communication asset library view

    Message type

    A design object that is used to insert either a text or graphic message within a message frame in Designer, for use in Content Author communications.

    Common panel in Design asset library view

    Messaging

    A type of design layout that is used in Designer documents to create designs of plain text output for SMS text messages.

    		Designer panel in Design asset library view

    Metadata (Class)

    An identification object that is used to sort and classify content in the asset library, and to easily identify resources in output files for use in downstream business processes.

    Common panel in Design asset library view

    Named flow

    A resource that identifies flow frames which are used in a targeted flow for one or more page designs within a Content Author communication or document template. For more information about named, targeted flows, see section 18 "Designing for Content Author" in OpenText Communications - Designer User Help (CCCD250100-H-UGD)section 18 "Designing for Content Author" in OpenText Communications - Designer User Help (CCCD250100-H-UGD).

    Common panel in Design asset library view

    Notification model

    A resource that lets you create a notification flow based on an orchestration flow instance and a flow step event. Notification flow models contain nodes that receive alerts from an alert node based on an event queue, then they process the rest of the flow like a regular flow model.

    Notification models are configured in Orchestrator.

    Flow panel in Orchestration asset library view

    Output

    A resource to which you can assign electronic and print output drivers.

    Delivery panel in Communication asset library view

    Output queue

    A resource that contains the settings for output production.

    Delivery panel in Communication asset library view

    Package

    A resource that is generated when you upload an application package or subpackage to DAS from Design Manager. Each application package or subpackage that you upload generates a separate package resource in DAS.

    Note

    Content Author supports only subpackages that contain all campaigns exclusively or a single document exclusively.

    Packages panel in Orchestration asset library view

    Page

    A type of design layout that is used in Designer documents to create designs for page-based output.

    Designer panel in Design asset library view

    Paper bin

    A resource that specifies the paper types available for print-based content when using a bin configuration for multiple paper types.

    Delivery panel in Communication asset library view

    Paper type

    A layout object that is used to define page characteristics for page designs in Designer communications.

    Common panel in Design asset library view

    PDF

    A PDF that can be imported into DAS for the purpose of converting its contents into Designer communications.

    Common panel in Design asset library view

    Recipient profile

    Optional delivery settings that are used to send copies of a communication to recipients other than the customer.

    Delivery panel in Communication asset library view

    Rule

    		 Rules allow you to include or exclude content or communications based on customer data. Rules that are saved as library rules in Designer appear as DAS resources.

    Data panel in Communication asset library view

    Run definition

    Contains information that is required to run the OpenText Communications engine and preview output for Content Author and Designer communications. An engine run definition is created either when a Design Manager application that contains a design pack is packaged to DAS or when a communication set is created.

    Run definitions panel in Orchestration asset library view

    Sample file

    A data file that contains sample input data that could be sent to the OpenText Communications engine to produce output. Sample files are used to map data sources and to populate variables in communication previews and simulations.

    Common panel in Design asset library view

    Search key

    A resource that lets you add data to the print stream for both electronic and print output.

    Delivery panel in Communication asset library view

    Section

    A design object that is used in Content Author communications.

    Author panel in Design asset library view

    Style

    A design object that defines characteristic formatting, which affects the appearance of documents. Styles are not implemented until they are part of a style sheet.

    Common panel in Design asset library view

    Style sheet

    A collection of style objects which can be applied to page, email, and web designs in Designer.

    Common panel in Design asset library view

    Template

    Contains the application content of a Design Manager application.

    A template is created when a Design Manager application that contains a design pack is packaged and uploaded to the DAS asset library from Design Manager, or when a Content Author template is created in Designer.

    Templates based on Design Manager applications are used to create new Content Author and Designer communications. Templates based on Designer communications are used to create new Content Author communications.

    In addition, templates based on Design Manager application packages that allow split communications in Content Author can function as Content Author communication container templates. And the templates for single-document subpackages that are linked to these application packages can function as Content Author document templates.

    For more information, see Section 8.2.1.1 “Generating templates for Content Author communication containers and documents”.

    Templates panel in Orchestration asset library view

    Text message

    A design object that is used in Content Author communications.

    Author panel in Design asset library view

    Variable

    DAS variables are created in the data source editor in Mapping view or the Data panel of the Communication asset library panel. They can be used for mapping in a data source and as design objects in a communication.

    Data panel in Communication asset library view

    Web design

    A type of design layout that is used in Designer documents to create designs for HTML5 output.

    Designer panel in Design asset library view
