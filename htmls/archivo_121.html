3.9.3   Searching text in multiple content resources

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    You can find content resources that contain the search text in their name or description properties, and also filter lists of content resources based on their workflow state, quick tags, and update status. Searching and filtering can be done separately, or in the same operation.

    If the Search text within content option is enabled, then you can find resources that contain the search text in their editable content instead of their name or description properties. For example, you can find clauses that contain the search text in one or more paragraph locations. In addition, when you search for text within content, the results can include instances where the matching text appears in the standard variation, alternate variations, or both.

    This section includes the following procedures:

    • Searching in content resource names or descriptions

    • Searching within editable text in multiple content resources

    • Filtering content resources using advanced search options

    Note

    If the Search text within content option is enabled, you can also replace text in editable text content.

    Searching in content resource names or descriptions

    1. In the Search box, enter the search text.

    2. Click the search button . The search results list all content resources that contain the search text in their name or description.

    3. Optional In the Search box, click the arrow, and then do the following in the Advanced search options dialog box:

      • Ensure that the Search text within content check box is cleared.

      • Select any advanced search options.

    4. Click Search.

      In the Search box, the search button changes to the advanced search button to indicate that advanced search options are selected.

    5. To search again with the same options selected and new search text, enter the new search text in the Search box and then click the advanced search button .

      Any advanced search options that you select apply to your searches until you change the selection.

    Searching within editable text in multiple content resources

    1. In the Search box, enter the search text, and then click the down arrow.

      You can specify search text that appears within the standard variation or in any alternate variations of the content resources. For example, if some content resources have locale-specific text in their alternate variations, you can find resources that contain the locale-specific text.

    2. In the Advanced search options dialog box, select the Search text within content check box.

      Note

      If you do not see the Search text within content check box, then ask your administrator about adding this option.

    3. Optional Select other advanced search options.

    4. Click Search.

      In the Search box, the search button changes to the advanced search button .

      Any advanced search options that you select apply to your searches until you change the selection.

    Filtering content resources using advanced search options

    1. In the Search box, optionally enter the search text, and then click the down arrow.

      Tip

      Entering search text allows you to search and filter content resources in one operation.

    2. Optional In the Advanced search options dialog box, select the Search text within content check box if you want to search and filter resources that contain the search text in any of their variations.

    3. In the Advanced search options dialog box, select one or more check boxes in the following filter areas:

      • Workflow state

      • Quick tags

      • My objects

    4. Click Search.

      In the Search box, the search button changes to the advanced search button to indicate that advanced search options (filters in this case) are selected.

    5. To search again with the same filters selected and new search text, enter the new search text in the Search box and then click the advanced search button .

      Any advanced search options that you select apply to your searches until you change the selection.
