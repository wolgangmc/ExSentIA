3.8.4   Replacing all search results while editing

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    1. In an editing view, search for the text that you want to replace.

    2. In the search bar, enter your replacement text in the Replace with box.

      Tip

      Before replacing text in Section view, see Section 3.8.5 “Guidelines for searching and replacing text in Section view”.

    3. In the search bar, click Replace all to replace all search results in all content in the current editing view.

      Note

      You can replace results in a piece of content only if the content meets the requirements for editability.

    4. For Section view: In the dialog box that displays the message, Are you sure you would like to replace all selected results?, click Replace.

      Important

      If you replace text in multiple canvases using Replace all, you cannot undo replacements because each piece of content is automatically saved after replacements are applied. Keep in mind that results are replaced only if the content that meets the replacement prerequisites.

    5. Optional To start a new search, in the search bar, enter new search text in the Search for box, and then repeat step 2 through step 4 to replace all search results.

    6. To close the search bar, click Stop.
