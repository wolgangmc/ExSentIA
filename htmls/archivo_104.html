    OpenText Communications resources are stored in the tenant-specific DAS database and can be managed using the asset library views in the OpenText Communications web client. The resources that are available in the asset library are exclusive to the OpenText Communications tenant that you are signed into, and are visible only if they are also associated with the domain in which you are working. Your user role and the permissions that are assigned to that role determines which views you can access. If you have questions about your access privileges, talk to your system administrator.
