4.2.3.5   Searching for a specific data element

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    If you want to search for a specific data element, type any part of the element name in the Find element box in the Data Source Editor.

    As you type, any data elements with names that match the entered text are highlighted. The number of matching results is listed on the right side of the Find element box.

    To navigate through the search results, use the up and down arrows, or you can press ENTER to move down and SHIFT+ENTER to move up.

    To clear the search results, click X.
