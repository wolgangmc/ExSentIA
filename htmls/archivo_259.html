    You can use the following keyboard shortcuts in Rationalization as an alternative way to perform some common tasks that are typically performed using a mouse or another input device.

    Keyboard shortcut Action
    ALT+F1 Open Rationalization Help.
    CTRL+Z Undo the previous action.
    CTRL+Y Redo the previous action.
    TAB In selection mode, move focus to the next object.
    SHIFT+TAB In selection mode, move focus to the previous object.
    ENTER In selection mode, select the current object.
    CTRL+DOWN ARROW In selection mode, with a text or image fragment selected, scroll down the list of fragments.
    CTRL+UP ARROW In selection mode, with a text or image fragment selected, scroll up the list of fragments.
    CTRL+CLICK

    Select multiple fragments in the list and combine them into one fragment by providing a new name and saving.

    Note that you cannot select multiple variables at the same time.
