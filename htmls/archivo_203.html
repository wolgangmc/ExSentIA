5.14   Using style sheets to enforce corporate styles

    You can use style sheets to help enforce text formatting styles and standards. Style sheets are similar to templates, but instead of guiding and standardizing page layout, style sheets help control the consistency of text styles and formatting. Using style sheets is one way to ensure that all outgoing text meets corporate styles and standards, which is especially important if you have a branded corporate design.

    You can use the following types of style sheets:

    • Design pack-based style sheets – Design pack-based style sheets are style sheets created by OpenText Communications Design Manager administrators. These style sheets contain an individual set of styles. If you want to select different styles for different customers, you use a selection list variable to reference other design pack-based style sheets in order to select alternate style sheets. Each alternate style sheet is treated as a different resource. For more information about configuring design pack-based style sheets, see section 2.4.3 "Style sheets" in OpenText Exstream Design and Production - Installation, Upgrade, and Administration Help (CCDPICU250100-H-IGD)section 2.4.3 "Style sheets" in OpenText Exstream Design and Production - Installation, Upgrade, and Administration Help (CCDPICU250100-H-IGD).

    • Communication set-based style sheets – Communication-set based style sheets are style sheets that you can create in the Design asset library. These style sheets contain a default set of styles and can optionally contain alternate style sheet information. You associate the default style sheet with a variable that contains the static selection list that you want to use to control style sheet selection. Each alternate style sheet is associated with a value from that static selection list. If you want to select different styles for different customers, you associate customers with the data for the specific alternate style sheet. Both the default style sheet and alternate styles are treated as a single resource. For more information about configuring communication set-based style sheets see Section 5.14.1 “Creating communication set-based style sheets”.

    After you have decided which type of style sheet that you want to use and created your style sheet objects, designers can associate style sheets with a design, apply styles to content, and preview style sheets or alternate style sheets in simulation. For more information about designing with style sheets, see section 13.1 "Enforcing corporate standards" in OpenText Communications - Designer User Help (CCCD250100-H-UGD)section 13.1 "Enforcing corporate standards" in OpenText Communications - Designer User Help (CCCD250100-H-UGD).
