2.4.1.2   Creating configurations for Content Author

  • Dashboards view

    You can use the Configurations panel in Tenant management view to make the following configurations for Content Author:

    On the General tab:

    Workflow

    Select the Use default workflow check box to do the following:

    • Enable the default workflow in Content Author. The default workflow type in each domain—standard, simple, or advanced—depends on the domain's Workflow type property setting. For more information about the workflow type for a domain, see Section 2.3.3.1 “Workflow types in domains”.

    • Make the Workflow type setting for each domain available in Tenant management view.

      For more information about making workflow settings available, see Section 2.3.3.2 “Showing or hiding workflow settings in domains tenant-wide”.

    Clear the Use default workflow check box to do the following:

    • Disable the default approval workflow in Content Author only.

    • Make the Workflow type setting for each domain unavailable in Tenant management view.

    Note

    The tenant-wide setting Use default workflow affects Content Author only.

    If the Use default workflow check box is cleared, workflow state buttons in Content Author are still visible, but they are view-only.

    The workflow in each domain—standard, simple, or advanced—is always available in Designer and in the asset library views.

    You might want to disable the default workflow for Content Author if your deployment is integrated with a third-party workflow tool. In that case, workflow changes happen outside of Content Author, but still appear in Content Author.

    For more information about workflow types, see Section 2.3.3.1 “Workflow types in domains” and Section 3.6 “Managing approval workflows”.

    Effectivity

    You can select the Use effectivity check box if you want to let business users apply effective dates to messages, clauses, and sections in Content Author. You can also set effective and expiration dates on messages, clauses, and sections in Design asset library view; however, business users will be able to change that setting later on in Content Author.

    On the Dictionaries tab:

    You can use the settings on the Dictionaries tab to manage installed spelling dictionaries for Content Author.
