8.3.2.2   Working with migration sets

  • Dashboards view

    A migration set is a DAS resource that contains PDF, DOCX, or StreamServe content to be imported using the rationalization process. The imported content is then available for use in Designer and Content Author communications.

    • For communications that use a DAS data source with one or more linked migration sets, those migration sets are also linked to the communication. You can link up to five migration sets to a data source.

    • For communications that use a design pack, you can link up to five DAS migration sets directly to the communication.

    In Designer and Content Author, you can then insert the text fragments, images, variables, and rules from the associated migration sets in your designs.
