5.11   Creating paper types

    You can use the Common panel in the Design asset library view to create paper types for use with your communications. A paper type resource is used to define the size, and color of paper stock. Paper types can also define the screen properties for electronic output.

    At least one paper type is required to create a page design in Designer. Designers can select paper types that are created in Design asset library view as well as paper types that are defined in Design Manager applications.

    To add a new paper type:

    1. In the Common panel, click Add, then click Paper type in the selection list.

    2. In the Properties panel, provide the following information:

      • Enter a name and, optionally, a description for the paper type resource.

        Names do not support the following special characters: & / : * ! ? > < | + # %

      • To define the dimensions of the paper type, enter the appropriate values in the Width box and Height box. To change the units of measurement, go to > Settings, and select the desired unit in the Units of measurement list.

      • To specify a background color for the paper type, click the Color selection box, choose a color from the color palette, and then click Apply.

    3. Click Create.

    Defining paper type properties

    After you create the paper type object, you can use the Properties panel to view and modify the following properties.

    1. For Postscript output, in the PostScript media name box, specify the name of the media type to use with the setpagedevice command. Media names define special settings for your printing system and the paper stock you use for a particular print run. The maximum length is 60 characters.

    2. For AFP output, complete the following steps as needed:

      1. In the AFP copygroup name box, specify a copygroup name to identify the paper type in the AFP print stream. The maximum length is eight characters. The copygroup name you specify is included as a part of the Invoke Media Map (IMM) command to request the paper type in the AFP formdef.

      2. If the printer must use the same paper type, regardless of different pages used in a design, you can specify a suffix to inform the printer of any page rotation. To apply a suffix, complete the following steps as needed.

        1. To apply a suffix that represents the page orientation, select Add orientation suffix (P, L, I, J). When this option is selected, one of the following suffixes is applied: P (portrait), L (landscape), I (portrait reversed), or J (landscape reversed).

        2. To apply a suffix that represents whether the page is simplex or duplex, select  Add simplex/duplex suffix (S, D). When this option is selected, one of the following suffixes is applied: S (simplex) or D (duplex).

      3. To specify that a page is an N-up instead of a larger page, select Generate as N-up page in copygroup. If you select Generate as N-up page in copygroup, use the N list to specify the value for N.

      4. In the Presentation Device Media Source drop-down list, select a presentation device media source. The options available in the list include different media sources that can be selected in the Medium Modification Control (MMC) structured field for the copygroup.
