9.4.5   Viewing a flow instance graphic

    For each flow instance, the General tab of the Properties panel provides a graphical representation of the flow model that is associated with that flow instance. You can use this visualization to quickly identify the progress of an orchestration job, pinpoint any errors, or check the job route.

    You can interact with the visualization in the following ways:

    • Selecting a node in the visualization opens the steps for that node in the Run history section.

    • Selecting a flow step in the Run history section highlights the corresponding node in the visualization.

    • Selecting Show route for a flow step in the Run history section shows the nodes that are involved in that route in the visualization.

    • Hovering over a node displays its status.
