    Although the following objects are typically created and defined from Designer, they are also stored in the web client on the Designer tab of the Design asset library. From the web client, you can review the settings for these objects and define how these objects behave when they interact with customer data and output.

      • To view or edit a rule – Click the Rule action button and select Edit/View.
