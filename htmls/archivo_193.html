5.4   Adding a DOCX file

    DOCX files can be stored in the asset library as a resource. These DOCX files can then be included as documents in Designer as needed.

    To add a DOCX file to the asset library:

    1. In the Design asset library, click the Common tab.

    2. Click Add > DOCX.

    3. In the Browser window, select the DOCX file that you want to add to the library, and click Open.

    4. After the file loads, you receive a validation message.

      • If the upload is successful, click Close and continue to step 5.

      • If the upload is unsuccessful, repeat step 3 through step 4 and select a different document.

    5. Optional After you have created the DOCX object, you can define the following common object settings:

      • Domains – For information about defining the domains to which a resource is assigned, see Section 3.13 “Associating resources with domains”.

      • Workflow – For information about managing an approval workflow, see Section 3.6 “Managing approval workflows”.

      • References – For information about viewing the linked resources connected to a selected resource, see Section 3.3 “Viewing resources in asset library views”.

    After you have completed the setup of the DOCX object in the asset library, the DOCX can be used to create a new document object in Designer and you can specify how the content is divided into clauses for inclusion and reuse.
