3.8.2   Prerequisites for replacing text while editing

  • Communication asset library view
  • Design asset library view
  • Orchestration asset library view
  • Monitor view
  • Rationalization asset library view
  • Dashboards view

    You can replace text in content if it meets the following requirements:

    • You have editing rights for the content.

    • The content is in the draft workflow state.

    • The content is not reserved by another user.

    In Versions view, you can replace text only in the latest version, and only if the latest version meets the requirements above.

    Tip

    You can replace text in a clause in Section view even if the section is not editable.
