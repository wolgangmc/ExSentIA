8.1.2   Starting a Designer communication

    You begin in Communication launcher view in the OpenText Communications web client by selecting from a list of existing communications or by creating a new communication from a list of available application templates or communication sets.

    Note

    Only approved communication sets are available to create communications.

    To access Communication launcher view, click the Communication launcher button on the application toolbar in the OpenText Communications web client.

    If you have questions about the design packs that are available, talk to the person or people who creates Design Manager applications in your OpenText Communications environment.

    Creating a new communication

    1. In the Communications panel, click the New button to see a list of available templates and communication sets.

    2. In the Source column, select a template or communication set, and click Next.

    3. If prompted, in the Data source name column, select a data source, and click Next.

    4. Specify a name and, optionally, a description for the communication that you want to create.

      Names do not support the following special characters: & / : * ! ? > < | + # %

    5. In the Category tags applied area, click on the tags that you want to apply. To apply all tags, click Add all.

      You can also use the Search box to search for a specific tag by name.

      Note

      This option might not be available in your environment. If you have questions, speak to your administrator.

    6. Click Create.

      The new communication opens in the Communication Composer in Designer.

    Opening an existing communication

    1. In the Communications panel, click one of the following buttons:

      • Library—Displays all of the existing communications

      • Recent—Displays a list of communications that you have recently opened in Designer

      • Favorites—Displays a list of communications that have been bookmarked

        Tip

        If you work frequently with a specific communication, you can bookmark that communication so that it appears in the Favorites list. To bookmark a communication, point to the communication in the Library or Recent tabs, and click the Bookmark icon in the Favorites column.

    2. Select the communication that you want to open, and click Open.

      The communication opens in the Communication Composer in Designer.

      Note

      Users that have only Approver permission initially see only communications in the Submitted for approval workflow state when they access the Communications panel. If those users want to see communications in other workflow states, they must apply an advanced search filter that has the other states selected. For more information, see Specifying advanced search options.

    Setting measurement units and spelling dictionary language

    1. In Communication Composer, select a design, component, or section and then click Open.

    2. In Design view, click the hamburger menu , and then click Settings.

    3. In the Settings dialog box, make selections from the following lists as needed:

      • Units of measurement—This is the default unit of measure used when working with content in Design view. You can specify inches, centimeters, points, or pixels.

      • Spelling dictionary—This is the default language used for spell-checking in Design view. If you have questions about the dictionaries that are available, speak to your system administrator.

    4. Click Save.
