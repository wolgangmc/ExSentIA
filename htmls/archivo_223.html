    The following table summarizes the differences between Content Author communications and Content Author communication containers:

    Content Author communications

    Content Author communication containers

    Communications cannot include Content Author documents. Communication containers include one or more Content Author documents.

    Communications include designs, such as pages, clauses, and messages, directly in the communication.

    Communication containers include designs only in their associated Content Author documents.

    Communications can have content frames that are editable in Content Author.

    In communication containers, only documents can have content frames that are editable in Content Author.

    Communications are created using templates based on Design Manager applications or Designer communications.

    Communication containers are created from container templates and document templates. These templates are generated from specially configured Design Manager application packages and their linked single-document subpackages.

    For more information, see Section 8.2.1.1 “Generating templates for Content Author communication containers and documents”.
